# py4web + Caddy + MySQL + Fail2Ban en Docker

## 1️⃣ Requisitos
- Docker
- Docker Compose
- Git

## 2️⃣ Instalación
```bash
git clone https://github.com/TU_USUARIO/py4web-docker.git
cd py4web-docker
docker compose up -d --build
```

## 3️⃣ Actualización
```bash
git pull
docker compose up -d --build
```
